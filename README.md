# Job Description Skill Extractor

A Generative AI application that extracts:

- Skills
- Experience
- Education

from Job Descriptions using Gemini LLM.

## Technologies Used

- Python
- Streamlit
- LangChain
- Gemini AI
- Pydantic

## Features

- Structured JSON output
- Prompt Engineering
- Output Validation
- Streamlit UI
